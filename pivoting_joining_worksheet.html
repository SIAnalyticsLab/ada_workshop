Purpose. Recognise wide and long data, then reshape datasets with pivot_longer() and pivot_wider().

A Wide versus long

temperatures <- data.frame(
  country = c("Sweden", "Denmark", "Norway"),
  avgtemp.1994 = 1:3,
  avgtemp.1995 = 3:5,
  avgtemp.1996 = 5:7

2. “In wide data, repeated measurements are often stored across multiple columns.”
3. “In long data, an observational unit may occupy multiple rows.”

B Pivot wide to long

infant_deaths_wide %>% 
  pivot_longer(cols = x2010:x2015)

C Use informative names

infant_deaths_wide %>% 
  pivot_longer(
    cols = x2010:x2015,
    names_to = "year",
    values_to = "deaths_count"
  )
Hint options: use pivot_longer(), names_to, and values_to.

D Clean pivoted names

infant_deaths_wide %>% 
  pivot_longer(
    cols = x2010:x2015,
    names_to = "year",
    values_to = "deaths_count"
  ) %>% 
  mutate(year = parse_number(year))
